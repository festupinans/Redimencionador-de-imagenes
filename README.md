# Resize Image Project

Este proyecto es una aplicación de Python con interfaz gráfica para seleccionar imágenes y redimensionarlas. El tamaño de salida de cada imagen será el más cercano al original que sea divisible por 4 (alto y ancho por separado). Las imágenes redimensionadas se guardan en una subcarpeta llamada 'resize4' en la misma ruta de la imagen original.

## Requisitos
- Python 3.8+
- Paquetes: Pillow, tkinter

## Uso
1. Ejecuta el archivo principal.
2. Selecciona una o varias imágenes.
3. Las imágenes redimensionadas se guardarán en la carpeta 'resize4' junto a la original.
